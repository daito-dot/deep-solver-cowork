# First Principles Thinking（第一原理思考）

## 概要
既存の前提・慣習・類推を全て疑い、物事を最も基本的な真実まで分解してから再構築する思考法。アリストテレスに由来し、現代ではイーロン・マスクの意思決定手法として広く知られる。

## 核心プロセス

### Phase 1: 前提の解体（Deconstruction）

**現状の「常識」を全てリストアップする:**
1. この問題について「みんなが知っている」ことを列挙
2. 各項目について「なぜそうなのか？」を問う
3. 根拠が「前からそうだから」「みんなそう言っているから」「専門家がそう言うから」の場合、**それは前提であって事実ではない**

**前提の分類:**
| 分類 | 定義 | 扱い |
|---|---|---|
| 物理法則 | 熱力学、量子力学等に基づく | 疑わない（通常は） |
| 論理的必然 | 数学的証明、定義上の真理 | 疑わない |
| 経験的事実 | 実験・観測で繰り返し確認済み | 条件付きで信頼 |
| 制度的前提 | 法律、規制、業界慣行 | 変更可能。疑え |
| 文化的前提 | 「普通はこうする」 | 最も疑うべき |
| 技術的前提 | 「現在の技術ではこれが限界」 | 時間とともに変わる。疑え |

### Phase 2: 基本真理の特定（Foundation）

解体した後に残る「これ以上分解できない真実」を特定する:
- 物理的に何が可能で何が不可能か？
- この問題の本質的な目的は何か？（手段ではなく目的）
- 関係者が**本当に**必要としているのは何か？（表明された要求ではなく、根本的欲求）

### Phase 3: 再構築（Reconstruction）

基本真理だけを出発点として、ゼロから解決策を構築する:
1. 「もし何の制約もなければ、この目的を達成する最も直接的な方法は何か？」
2. その方法を阻むのは何か？（物理法則か？制度か？お金か？）
3. 阻む要因のうち、本当に動かせないものはどれか？
4. 動かせる要因を動かした場合、何が可能になるか？

## 典型的な罠

### Analogical Reasoning との混同
- First Principles: 「バッテリーは高い」→ なぜ？ → 原材料コスト → 原材料を直接調達すれば安くなる
- Analogical: 「バッテリーは高い」→ 他社も高い → 高いのが普通 → コスト削減は難しい

### 過度な解体
- 全てを物理法則まで分解する必要はない。分解は「行動可能な洞察が得られるレベル」で止める
- 「水は H2O からできている」まで遡る必要があるのは、水の性質自体が問題の核心である場合のみ

### 再構築の飛躍
- 基本真理から解決策への道筋は、論理的に追跡可能でなければならない
- 「全ての前提を疑った結果、直感的にこう思う」は First Principles ではない

## Deep Solver での使い方
- **Abstractor のメタ層分析**で活用: 「なぜこの課題が課題なのか」の前提を解体する
- **Dialectic の Antithesis 構築**で活用: Thesis の前提を攻撃する根拠として
- **Orchestrator のフレームワーク選択**で活用: 前例の有無スコアが高い（前例なし）課題に適用
