# 未知課題の解体ヒューリスティクス

未知の課題（Unknown Unknowns）に直面した際、問題を「計算可能な状態」に変換するための3つの根本的操作。

## 1. トポロジー変換

課題の表層（ドメイン固有の専門用語・文化的コンテキスト・感情的色彩）を完全に剥ぎ取り、純粋な構造に変換する。

### 手順
1. **エンティティ抽出**: 課題に関わる全ての実体をノードとしてリストアップ
2. **関係抽出**: ノード間の関係をエッジとして定義（因果、依存、対立、共生、競合など）
3. **フロー特定**: 何が（情報、資源、権力、感情）どの方向に流れているかを特定
4. **ボトルネック発見**: フローが滞留・集中している箇所を特定
5. **グラフ特性の分析**:
   - 中心性（最も影響力の大きいノード）
   - クラスタリング（密結合するグループ）
   - ブリッジ（グループ間を繋ぐ重要なノード/エッジ）
   - サイクル（循環構造 = フィードバックループの候補）

### 適用例
「都市の交通渋滞」→ ノード（交差点、駅、商業施設）、エッジ（道路、路線）、フロー（車両、歩行者）、ボトルネック（主要交差点の容量超過）

## 2. 不変量の特定

環境や条件がどう変化しても**絶対に変わらない物理的・論理的制約**を特定し、それをアンカー（碇）とする。

### 手順
1. **物理的不変量**: エネルギー保存、情報の処理限界、物質の有限性、時間の不可逆性
2. **論理的不変量**: 矛盾律、ゲーデルの不完全性、計算量の理論的下限
3. **人間的不変量**: 認知の限界（ミラーの法則 7±2）、注意資源の有限性、死の不可避性
4. **制度的不変量**: 慣性（既存制度の変更コスト）、合意形成の困難さ、情報の非対称性

### 各不変量の評価
- **硬度**: 本当に不変か？どの条件下で破れるか？
- **関連度**: この不変量は課題の核心にどれだけ関わるか？
- **利用可能性**: この不変量を解決策の「てこの支点」として利用できるか？

## 3. パラドックスの抽出

課題が未解決である最大の理由 — 「Aを立てればBが立たず」という核心的トレードオフを特定し、その解消を図る。

### パラドックスの類型
| 類型 | 構造 | 解消の方向 |
|---|---|---|
| **リソース競合** | AとBが同じリソースを奪い合う | リソースの増大、時間分離、代替リソースの発見 |
| **目標対立** | AとBが本質的に矛盾する目標 | 上位目標の発見、スコープの再定義 |
| **スケール矛盾** | 小規模では機能するが大規模では破綻（またはその逆） | 階層化、分散化、相転移の活用 |
| **時間矛盾** | 短期的に良いことが長期的に悪い（またはその逆） | 時間割引の再設計、段階的移行、可逆性の確保 |
| **認識矛盾** | 解決に必要な情報が、解決するまで入手できない | 仮説駆動アプローチ、段階的開示、プロトタイピング |

### 次元上昇による解消
パラドックスが解消不能に見える場合、**問題の次元を1つ上げる**:
- **2D→3D**: 平面上で衝突する2つのパスを、立体交差にする
- **静的→動的**: 同時に両立できないなら、時間的に交互に切り替える
- **個体→集団**: 1人では矛盾するが、集団で役割分担すれば両立する
- **確定→確率**: 確実な回答を求めず、確率分布として扱う
